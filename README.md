# 📚 PodgotviMe Numbers

Мобилно Android приложение за подготовка по математика за ученици от 7. клас.

Проектът е разработен като курсова работа по дисциплината **Мобилни приложения**.

## 📱 Основни функционалности

- Регистрация на ученик
- Вход в системата
- Автоматично вписване чрез JWT Token
- Изход от профила
- Връзка с Node.js REST API
- PostgreSQL база данни

---
